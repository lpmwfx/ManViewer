# ManViewer

Slint GUI viewer for `man/` documentation generated by [rulestools-documenter](https://github.com/lpmwfx/RulesTools/tree/main/crates/documenter).

## Usage

```bash
# Generate man/ docs for a project
rulestools gen /path/to/project

# Open the viewer
man-viewer /path/to/project
```

## Install

```bash
cargo install --git https://github.com/lpmwfx/ManViewer
```
